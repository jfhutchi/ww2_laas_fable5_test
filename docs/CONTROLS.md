# Controls

## Global

| Input | Action |
|---|---|
| Tab | Switch tactical ↔ direct tank control (selected tank, else nearest living Sherman) |
| Esc | Pause menu / resume |
| Space | Pause / unpause simulation (tactical mode) |
| F3 | Performance/debug HUD |
| P | Print current camera pose + seed to the console (`?cam=` format) |
| 1–9 | Recall control group (units) — falls through to camera bookmark |
| Ctrl+1–9 | Assign current selection to control group |

## Tactical command mode

| Input | Action |
|---|---|
| Left click | Select unit (clicking a roster row or unit marker also selects) |
| Shift + left click | Add/remove from selection |
| Left drag | Selection rectangle |
| Right click | Move selection — on a spotted enemy: attack that target |
| Right click (minimap) | Move selection to map point |
| A | Arm attack-move, then left click the destination |
| G | Arm attack-ground (HE area fire), then left click the point |
| S | Stop |
| H | Hold position |
| W/A/S/D or arrows | Pan camera |
| Edge pan | Push cursor to a screen edge |
| Q / E | Rotate camera |
| Mouse wheel | Zoom |
| Middle-mouse drag | Rotate / tilt camera |
| Double-click roster row | Center camera on unit |

Command panel buttons mirror A/G/S/H plus TAKE CONTROL (Tab).

## Direct tank control

| Input | Action |
|---|---|
| W / S | Throttle forward / reverse (heavy acceleration model) |
| A / D | Steer tracks (pivots in place at low speed) |
| Shift | Overdrive |
| Mouse | Aim turret / cannon (turret chases the reticle) |
| Left click | Fire main cannon (when reloaded and turret aligned) |
| Right click (hold) | Fire hull machine gun |
| R | Reload is automatic — R is acknowledged but not required |
| C | Cycle camera shoulder/distance preset |
| Tab / button | Return to command (tank reverts to tactical AI hold) |

While you drive one tank, every other unit keeps obeying its tactical orders and the battle continues.
